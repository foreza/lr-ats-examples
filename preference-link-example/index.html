Hello there!

This page features LiveRamp's GDPR CMP with the capability of storing data / pulling data from PL Enterprise.

This example assumes that PL Enterprise is the single source of truth for consent strings, and that a unique subject ID is being used.

Check your network inspector to see the API calls being made!

The example workflow is as follows:

  • Once the page is loaded, we'll suppress the loading of the CMP dialog (regardless of whether we have consent already)
  • Call PL Enterprise for the subject ID once we have it.
  • If the subject ID does NOT yet exist within PL enterprise; show the CMP dialog.
  • If the subject ID EXISTS within PL enterprise, use the CMP SDK's ability to set the consent string manually (leveraging TCF IAB's libraries)
